# Medialist

Data structure

**medialist**

```js
{
  createdBy: 'userId',
  createdAt: 'timestamp',
  name: '',
  purpose: '',
  slug: '',
  contacts: ['contactSlug']
}
```


**contacts**

```js
{
  createdBy: 'userId',
  createdAt: 'timestamp',
  medialists: {
    'medialistSlug': 'status'
  },
  name: 'Jane Smith',
  twitter: 'janesmith',
  jobs: [{
    title: 'Contributor',
    org: 'PR Public Relations'
  }],
  bio: 'Lorem ipsum...',
  emails: [{
    address: 'jane.smith@pr.com',
    type: 'work'
  }],
  phones: [{
    number: '01234567890',
    type: 'work'
  }],
  slug: 'janesmith' // Twitter handle if available, otherwise autogenerated
}
```

**feedback**

```js
{
  createdBy: 'userId',
  createdAt: 'timestamp',
  message: 'Spoke to @janesmith about #medialist, she loves it',
  contacts: ['janesmith'],
  medialists: ['medialist'],
  status: 'status'
}
```
